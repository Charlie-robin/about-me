Well Designed Sites

NixOS:

One of the reasons I like this website is due to how rare it is for this style of Linux distro to have such a good looking and usable website. It's easy to find everything you may want to do and find out initially, and its clear how it could be useful for returning users looking for documentation/support etc. As a comparison, the Arch Linux Website is, in my opinion, pretty bad. NixOS is quite a niche distribution which will largely only be used by Linux enthusiasts, similar to Arch. However, unlike Arch, the website has been designed to have a really good UX and be welcoming to the distro.

Rust Lang:

The rust-lang website is very well designed, it is clearly laid out, with the most likely things the user is looking for further up, and less used things such as maintainership and NPM funding further down. The download page is very refreshing, I setup linux quite often and rust is always one of the first things I install, having platform-agnostic install methods presented to the user immediately on the page is a big benefit. I also like how simple the website is, navigating it with purely the keyboard is quite easy. The site doesn't necessitate any specialised accessibilty methods due to how simple it is. Making the site better for able-bodied users as well as disabled users.
