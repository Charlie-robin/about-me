A really terrible circuit schematic builder
This was a school project where I had to complete a project that covered enough different things to get a good mark. So I chose a full stack web application using ASP.NET MVC with C#, I bit off way more than I could chew at the time and so didn't have much time to properly design it (or fix design shortcomings). I use Neovim as my main editor, which is famous for having poor Java support. At work we use both Java and the Cucumber testing framework which uses a language called Gherkin.
